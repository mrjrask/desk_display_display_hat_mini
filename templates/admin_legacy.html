Legacy configuration editor

This minimal editor keeps the classic JSON workflow. To enable the new playlist UI, set metadata.ui.playlist_admin_enabled to true in the configuration and reload.

You can also append ?legacy=1 to the URL to reopen this view at any time.

Schema quick reference

{
  "version": 2,
  "catalog": {"presets": {}},
  "metadata": {"ui": {"playlist_admin_enabled": false}},
  "playlists": {
    "main": {
      "steps": [
        {"screen": "date"},
        {"playlist": "weather"},
        {"rule": {"type": "every", "frequency": 3, "item": {"screen": "inside"}}}
      ]
    }
  },
  "sequence": [{"playlist": "main"}]
}
